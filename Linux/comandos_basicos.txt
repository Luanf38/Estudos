Comandos básicos:
    su = dá privilégios de root para o user logado

Comandos de navegação:
    cd = Navega entre diretórios
    ls = lista os arquivos e pastas da diretório atual
    ls -l = lista e mostra detalhes de pastas e arquivos
    . = diretório atual
    .. = diretorio pai
    / = diretorio root
    ~ = diretorio do usuario logado

Comandos de manipulacao de pastas e arquivos:
    mkdir = cria uma pastas
    rm = deleta um arquivos
    rm -r = deleta uma pasta
    vi = comando do editor de texto
    cat = le um arquivo
    tac = le um arquivo de tras para frente
    mv = move uma pasta ou arquivo, mas tambem renomeira
        mv arquivo destino (mover)
        mv arquivo_existente novo_nome (renomear)
    cp = copiar um arquivo
    > = direciona o inoput para o arquivo desejado
    chmod = altera as permissões do arquivo
    chmod +x arquivo = adiciona permissão de execussão ao arquivo

Comandos systemd
    systemd-analyze = mostra o tempo de carregamento do sistema e pode-se
        acrescentar o "blame" para detalhar o tempo de cada processo
    hostnamectl = mostra as informações do sistema
    systemctl = exibe o status de todos os serviços do servidor

Comandos de rede
    lspci = Mostra todos os dispositivos conectados ao servidor
    ip address = da informações sobre os dispositivos e endereços de rede
    ifdown "Nome da interface de rede" = reseta as configurações de ip do servidor
    ifup "Nome da interface de rede" = para pegar uma nova configuração de ip
