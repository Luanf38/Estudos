## Tipos de engenharia de software
#### Engenharia de Software Operacional
&emsp;A Engenharia de software no nível operacional concentra-se em como o software interage com o sistema(conjunto de hardware e software), se está ou não dentro do orçamento, a usabilidade, a funcionalidade, a confiabilidade e a segurança.

#### Engenharia de Software de Transição
&emsp;Esse tipo se concentra em como o software reagirá quando for alterado de um ambiente para outro. Normalmente, é preciso alguma escalabilidade ou flexibilidade no desenvolvimento.

#### Engenharia de Software de Manutenção
&emsp;Especifica o quão bem um software funciona no ambiente em mudança. Modularidade, manutenibilidade, flexibilidade e escalabilidade estão na parte de manutenção.
